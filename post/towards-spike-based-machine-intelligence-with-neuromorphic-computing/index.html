Abstract

此文章从算法和硬件两个方面来回顾基础的学习和硬件框架，讨论了主要面临的挑战和未来的方向，并重点放在算法-硬件框架上。

Main

首先介绍了为什么要研究类脑计算

  1. 今天，科学家们认为脑就像一个计算机，神经网络的兴起确实让人看到了效率的大幅提高，但一个问题是能耗
  2. 大脑完成这么多高效的活动只需要20W，而一个标准计算机仅仅完成识别动作就要250W，这源于脑的生物特性：广泛连接、结构化功能组织的层级和基于时间的神经元和突触
  3. 神经元是脑的基本计算和存储单元，它传递spike来进行信息传递，spike是时间序列的，离散的，降低能耗的同时，可以更大程度利用时间信息。

Nowadays：

现在，基于脑机制的神经网络涌现，并且取得了良好的效果，如modern deep-learning networks(DLNs)就是基于人脑层次结构和神经突触框架的启发，但是这种神经网络的硬件基本是硅晶体管硬件计算系统。

Questions：

人脑的计算原理和硅基计算机有鲜明的区别：

  • 人脑是存算一体的（计算在神经元，存储在突触），计算机是计算和存储分离的
  • 人脑中大量存在三维连通性，而计算机硬件多为二维连接
  • 晶体管主要构建确定性布尔电路开关，和人脑基于脉冲的事件驱动随即计算不同

尽管如此，在当前的深度学习革命中，硅基计算平台（例如图像处理单元（GPU）云服务器）已成为一个重要的贡献因素。

但是，使得“通用智能”（包括基于云服务器到边缘设备）无法实现的主要瓶颈是巨大的能耗和吞吐量需求。例如，在一个由典型的2.1Wh电池供能的嵌入式智能玻璃处理器（smart-glass processor）上运行深度网络，就会让处理器在25分钟内将电池消耗殆尽。

在人脑的指引下，通过脉冲驱动通信从而实现了神经元-突触计算的硬件系统将可以实现节能型机器智能。神经形态计算始于20世纪80年代晶体管仿照神经元和突触的功能运作（图2），之后其迅速演化到包括事件驱动的计算本质（离散的“脉冲”人造物）。最终，在21世纪初期，这种研究努力促进了大规模神经形态芯片的出现。

Main Discussion

Intelligence（algorithmic）

不同的SNN学习机制（基于脉冲或梯度的监督学习和非监督学习机制），重点讨论基于时空事件的表征。大量讨论集中在时间和相关应用领域

Computation（hardware）

analog computing（模拟计算），数字神经运动系统，都超越了当前的冯诺依曼结构。

Combination

最后，我们将讨论算法的硬件协同设计前景，说明算法具有用于对抗硬件漏洞的鲁棒性，可以实现能耗和精度之间的最佳平衡。

Algorithmic outlook

Spiking neural networks

神经网络分成三代：

  • McCulloch–Pitt感知机，它执行阈值运算并输出数字（1、0）。基于sigmoid单元或修正线性单元（ReLU）
  • 第二代神经元单元增加了连续非线性，使其能够计算一组连续的输出值。第一代和第二代网络之间的非线性升级在扩展神经网络向复杂应用和更深度的实现方面起着关键作用。
  • 第三代神经网络主要使用“整合放电”（integrate-and-fire）型尖峰神经元，通过脉冲交换信息

第二代和第三代的主要区别就是：

  1. SNN是二进制，所以只在有信号来的时候才处于活跃状态，更节能，DLN无论何时都处于活跃状态。
  2. SNN输入值只有0和1，也减少了很多数学上的点积运算。
