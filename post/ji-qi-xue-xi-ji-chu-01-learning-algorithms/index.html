1 What is learning?

In our long term study career, we might be confused, What is learning？Subjectivly, I think it was a period to get some knowledge, skills or experience.

But when define a machine learning algorithm, which means an algorithm that is able to learn from data, we need a clear definition of learning.

Mitchell(1997) provides a definition. There are three elements in his definition of learning: Tasks TTT, Experiences EEE and Performance measures PPP.

A computer program is said to learn from experience EEE with respect to（关于，对于）some class of tasks TTT and performance measures PPP, if its performances at task TTT, as measured by PPP, improves with experience EEE.

So what is those TTT,EEE,PPP about?

2 Tasks,Performance measures,Experiences

2.1 The Tasks, TTT

Learning ≠\neq​= Tasks

Learning is a process to attening the ability to perform the task.

Machine learning tasks are usually described in terms of（就……而言） how the computer system can process an example

example: features we get from some objects or events we hope the machine learning systems to process.（my version)

example: a collection of features that have been quantitatively measured from some object or event that we want the machine learning system to process.（original version）

such as accuracy, errorrate,etc. Performance Measure depends on different tasks.

2.3 The Experiences, EEE

Machine learning algorithms can be broadly categorized as unsupervised or supervised.

dataset: a collection of many examples.

Most of the learning algorithm can be understood as being allowed to experience an entire dataset.（在整个数据集中获取经验。这种表达方式值得学习）

Unsupervised Learning Algorithm

Training on dataset with lots of features and learn useful properties of the structure of this dataset. In the context of deep learning, we usually want to learn the entire probability distribution that generated a dataset.(This means we can know how this dataset generated), whether explicitly（显式地） as in density estimation（密度估计） or implicitly（隐式地）for tasks like synthesis or denoising, like clustering, which consists of diving the dataset into clusters of similar examples.

Supervised Learning Algorithm

Experience a dataset containing features, but each example is alse associated with a label or target.

Important

Unsupervised learning and supervised learning are not formally defined terms（术语）.
